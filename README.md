# github-client

## Environment Setup

```
install nodejs and then install vue cli using "npm install -g @vue/cli" on your system.
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
